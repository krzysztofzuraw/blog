This repository contains code & blog post for my [blog](https://krzysztofzuraw.com).

## How to run

Install dependencies:

```shell
npm run install
```

Run development server:

```shell
npm run dev
```

Run production build:

```shell
npm run build
```

Generate empty placeholders for post entry

```shell
npm run new:post
```
